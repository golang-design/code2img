# code2img

a carbon service wrapper

## API

```
POST golang.design/api/v1/code2img
{
    "code": "code string"
}
```

Response pure text (better for iOS shortcut):

```
https://golang.design/api/v1/code2img/data/images/06ad29c5-2989-4a8e-8cd2-1ce63e36167b.png
```
